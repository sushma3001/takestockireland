# TakeStock Database

MySQL schema for TakeStock Ireland.

## Modules done

- Users (login)
- Locations
- Employee

## Setup

```
mysql -u root -p < database/init.sql
```

## Structure

- schema/ - table definitions
- seed/ - sample data
- docs/ - ERD

## Tech

- MySQL 8.0
- Spring Boot (planned)
- JPA (planned)
